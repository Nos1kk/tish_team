# TISH STORE — Админ-панель: Полный план функционала

## 1. 📊 ДАШБОРД (Главная)
- Общая статистика сайта:
  - Количество пользователей (новые за день/неделю/месяц)
  - Количество посещений (сегодня, вчера, за неделю, за месяц)
  - Общий объём продаж ($)
  - Количество заказов (по статусам)
  - Количество активных чатов
  - Популярные товары (топ-5)
- Графики:
  - Посещения по дням (линейный график)
  - Продажи по дням (столбчатый график)
  - Новые пользователи по дням
- Последние действия (лог):
  - Новые заказы
  - Новые регистрации
  - Новые отзывы
  - NFT покупки

## 2. 🛍️ МАГАЗИН (Управление товарами)
### 2.1 Список товаров
- Таблица всех товаров с поиском и фильтрацией
- Колонки: ID, Фото, Название, Категория, Цена, Старая цена, Рейтинг, Продажи, Статус (опубликован/скрыт)
- Действия: Редактировать, Скрыть/Показать, Удалить

### 2.2 Создание/Редактирование товара
- Поля:
  - Название (текст)
  - Описание (textarea с форматированием)
  - Категория (выпадающий список: UI Kits, Шаблоны, Иконки, Иллюстрации, Брендинг, 3D, Motion)
  - Цена ($)
  - Старая цена (для зачёркнутой)
  - Бейдж (NEW, HOT, SALE, или пусто)
  - Загрузка фото/видео (перетаскивание файлов, до 10 изображений + 1 видео)
  - Автор (имя, роль)
  - Компонентов (число)
  - Версия (текст)
  - Формат (Figma, Sketch, PSD, и т.д.)
  - Список «Что включено» (динамический список)
  - Градиент/фон карточки (color picker или CSS gradient)
- Предпросмотр карточки в реальном времени

## 3. 🎮 NFT МАГАЗИН
### 3.1 Управление коллекциями
- Создание коллекции:
  - Название коллекции
  - Описание
  - Обложка (загрузка файла)
  - Дата начала/окончания продаж
  - Статус (активна/скрыта)

### 3.2 Управление кейсами
- Создание кейса:
  - Название
  - Цена кейса
  - Коллекция (привязка)
  - Загрузка обложки/анимации кейса
  - Список NFT внутри кейса (с процентами выпадения)
  - Статус (в продаже/закрыт)

### 3.3 Управление NFT
- Создание NFT:
  - Название
  - Редкость (Common, Rare, Epic, Legendary, Mythic)
  - Цена (рыночная стоимость)
  - Цвет (для кастомизации профиля)
  - Стикер/эмодзи
  - Загрузка анимации (GIF, MP4, Lottie JSON, или CSS-класс)
  - Загрузка изображения-превью
  - Привязка к кейсу (в каком кейсе выпадает)
  - Процент выпадения
  - Ограниченный тираж (опционально: макс. количество)
- Таблица всех NFT с фильтрами по редкости, коллекции, кейсу
- Массовое редактирование (изменить редкость, цену у нескольких)

### 3.4 NFT Маркетплейс (будущее)
- Управление листингами пользователей
- Комиссия с продаж (настройка %)
- Модерация (одобрение/отклонение)

## 4. 👥 ПОЛЬЗОВАТЕЛИ
### 4.1 Список пользователей
- Таблица: ID, Аватар, Имя, Username, Email, Уровень, TISHARA, Заказов, Дата регистрации, Статус
- Поиск по имени, email, username
- Фильтры: по уровню, по дате регистрации, по активности

### 4.2 Профиль пользователя (админский просмотр)
- Вся информация о пользователе
- Начисление/списание TISHARA (с указанием причины)
- Выдача достижений вручную
- Выдача скидок/промокодов
- Выдача NFT вручную
- Изменение уровня
- Верификация (вкл/выкл)
- Бан/разбан
- Просмотр заказов пользователя
- Просмотр NFT коллекции пользователя
- История действий пользователя

### 4.3 Начисление бонусов
- Форма:
  - Выбор пользователя (поиск)
  - Сумма TISHARA
  - Причина/комментарий
  - Тип: начисление/списание
- История начислений от админа

## 5. 📦 ЗАКАЗЫ
### 5.1 Список заказов
- Таблица: ID, Товар, Покупатель, Сумма, Статус, Дата, Действия
- Фильтры по статусу, дате, сумме
- Поиск по ID заказа, имени покупателя

### 5.2 Управление заказом
- Изменение статуса (pending → prepaid → in_progress → invoice_sent → paid → completed)
- Привязка к чату
- Комментарий админа
- Возврат/отмена заказа
- Начисление TISHARA/XP за завершённый заказ

## 6. 🏆 ДОСТИЖЕНИЯ
### 6.1 Управление достижениями
- Список всех достижений
- Создание нового:
  - Название
  - Описание
  - Иконка (выбор из набора или загрузка)
  - Условие (выбор из шаблонов: N покупок, N отзывов, N рефералов, уровень N, потрачено $N, и т.д.)
  - Награда XP
  - Награда TISHARA
  - Редкое (да/нет)
  - Статус (активно/скрыто)
- Редактирование существующих
- Удаление (с подтверждением)
- Предпросмотр карточки достижения

## 7. 💰 ПОДПИСКА (Tishara Club)
### 7.1 Настройка уровней подписки
- Для каждой из 4 категорий:
  - Веб, IT-разработка
  - Дизайн и графика
  - Видеопродакшн и моушн
  - Продюсирование и комплексный маркетинг
- Для каждого уровня (Старт, PRO, VIP, Амбассадор):
  - Условие (минимальная сумма покупок)
  - Кэшбек (%)
  - Плюшки (текст/описание бонусов)
  - Статус (активен/скрыт)

### 7.2 Статистика подписок
- Сколько пользователей на каждом уровне
- Общая сумма кэшбека выданного
- Прогнозы роста

## 8. 👫 РЕФЕРАЛЫ
### 8.1 Настройки реферальной системы
- Бонус за регистрацию по коду (TISHARA)
- Бонус за первую покупку реферала (TISHARA)
- Процент от покупок реферала (%)
- Максимум рефералов на пользователя

### 8.2 Статистика
- Общее количество рефералов
- Топ рефереров
- Общая сумма выданных бонусов
- График регистраций по рефералам

## 9. ⭐ ОТЗЫВЫ (Модерация)
- Список всех отзывов
- Фильтры: по товару, по рейтингу, по дате
- Модерация: одобрить/отклонить/удалить
- Ответ от магазина на отзыв

## 10. 📢 УВЕДОМЛЕНИЯ
- Создание уведомлений:
  - Для всех пользователей
  - Для конкретного пользователя
  - По группе (по уровню, по категории покупок)
- Типы: информация, акция, предупреждение, система
- Шаблоны уведомлений
- История отправленных

## 11. 🎨 НАСТРОЙКИ САЙТА
- Логотип и название
- Цвета и тема
- Баннеры и промо-блоки
- Включение/отключение разделов (NFT магазин, подписка, рефералы)
- SEO настройки (title, description, keywords)
- Промокоды (создание, активация, деактивация)

## 12. 📈 АНАЛИТИКА
- Посещения по страницам
- Воронка конверсии (посещение → просмотр товара → корзина → покупка)
- Среднее время на сайте
- Источники трафика
- Устройства и браузеры
- Популярные поисковые запросы
- Экспорт данных в CSV/Excel

## ТЕХНИЧЕСКИЕ ЗАМЕТКИ
- Все данные хранить в localStorage (для демо) с возможностью перехода на API
- Каждая секция админки — отдельный JS модуль
- Роли: super_admin (всё), admin (без настроек сайта), moderator (отзывы, заказы)
- Все действия логировать в историю
- Критические действия — с подтверждением (удаление, бан, массовые операции)
